# Aegis & Voss — Final Premium Law Firm Website

A polished React/Vite law firm website with a premium, breathable layout.

## Run

```bash
npm install
npm run dev
```

Then open the Vite localhost URL.

## Build

```bash
npm run build
npm run preview
```

## Note

Before using for an actual law firm, replace names, lawyer images, contact details, representative matters and numbers with verified client-approved information.
